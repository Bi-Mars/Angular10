Data Binding in Angular

  1. String Interpolation: {{ "vaue" }}
  2. Properties Binding: [property] = "Value"
  3. Event Binding: (event)="value"
  4. 2 way binding: [()] = "value"

Requirement

  1. Create Input field which UPDATES a property ("username") via 2-way-binding
  2. Output the username property via String Interpolation (inside paragraph below the input)
  3. Add a button which many only be cickled if the username is NOT an empty string
  4. Upon clicking the button, the username should be reset to an empty string

How did I approached?

  • I created an application from scratch.
  • Started with angular.json and took a good glance of the configuration.
  • Then I checked index.html and emptied all the content
  • Next stop was app.module.ts
  • I created Component called data-binding
  • Then, I created data-binding.component.ts and added 3 important part:
    • Import: Imported @Component decorator
    • Use @Component: configured - selector and templateUrl
    • Exported the class
  • It's time to let the root component know about new component.

    I registered newly created component inside app.module.ts

  • I created basic outline for the requirements: input field, button, paragraph.
  • Time for some variable and Text Interpolation.
  • Used 2-way data binding for inputtext to update userName value everywhere
  • Added property binding for "disabled" property of the button
  • Added event binding for "click" event of the button
  • Incrementally added logic in component
  • Added selector specifed in the component to app.component.html
